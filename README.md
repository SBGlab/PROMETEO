# PROMETEO
Repository to store the models, the data and associated code for PROMETEO

## Folders
- Models:It contains Matlab models
- PlateReaderData: It contains Plate Reader Raw Data, including hysteresis experiment and Biofilm quantification with Crystal Violet Assay
- ConfocalImages: Contains confocal microscopy images on static cultures to directly visualize spatial organization of TSECO
- HPLC: It contains HPLC raw data from Congo Red Experiments
